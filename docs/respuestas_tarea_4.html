  1. Cree un mapa interactivo con leaflet que muestre el Parque Nacional Corcovado, el Parque Nacional Piedras Blancas y la Reserva Forestal Golfo Dulce (pista: pueden encontrar las coordenadas usando google maps). Para cada sitio añada un marcador usando la función de leaftlet addAwesomeMarkers. El mapa debe ser similar a este:

 

# cargar paquete
library("leaflet")

# crear mapa vacio
ll_map <- leaflet()

# añadir capas
ll_map <- addTiles(map = ll_map)

# definir centro
ll_map <- setView(map = ll_map, lat = 8.534785, lng = -83.552219,
    zoom = 10)

# añadir marcadores
leaflet::addAwesomeMarkers(map = ll_map, lat = c(8.534785, 8.737077,
    8.699383), lng = c(-83.552219, -83.498093, -83.275528))
